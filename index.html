CSV-to-Go

Convert CSV to Go struct

This tool instantly converts CSV into a Go type definition. Paste a CSV structure on the left and the equivalent Go type will be generated to the right, which you can paste into your program. The script has to make some assumptions, so double-check the output. (And go fmt your code!)

Make sure to use 'Shift+enter' instead of just 'enter' to insert the '\n' character needed to parse the CSV.
